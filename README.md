# TroneUqac
Un site web qui permet d'évaluer les toilettes de l'UQAC.


## Utilisation
### Prérequis
Avant de lancer l'application web il faut d'abord python. Nous utilison python 3.11.9 . 
Pour installer les dépendances, exécutez cette commande : pip install -r requirements.txt .

### Lancement
Une fois les prérequis installés, il faut se mettre à la racine du projet, soit dans 
le fichier TroneUqac, puis exécuter la commande : python main.py .
